
Instructions
Для запуска на локальном сервере:
1.Распакуйте архив в любом месте на Вашем компьютере.
2. С помощью любого текстового редактора откройте файл settings.py и укажите свои настройки MySQL в секции "DATABASES".
3. Активируйте виртуальную среду. Для этого из директории \.venv\Scripts нужно выполнить команду activate.bat 
4. Выполните минрацию базы данных. Для этого после активации виртуальной среды перейдите в директорию appointment_booking_system и выполните команды:
python manage.py makemigrations
python manage.py migrate
После этого с помощью консольного или графического клиента можно визуально убедиться, что в базе созданы нужные таблицы.
Подробнее о миграции базы данных можно почитать здесь: https://docs.djangoproject.com/en/4.1/topics/migrations/
5. Создайте суперпользователя, который будет администратором Вашего приложения. Для этого выполните команду:
python manage.py createsuperuser
Подробнее о создании суперпользователя здесь: https://docs.djangoproject.com/en/1.8/intro/tutorial02/
6. Запустите сервер джанго с помощью команды:
python manage.py runserver
7. Откройте окно браузера и введите в адресную строку адрес, который выведется в консоли после запуска сервера(http://127.0.0.1:7000/)


Приложение предполагает два режима работы: 

------------------
АДМИНИСТРАТОР
------------------
1. Зайдите по адресу  http://127.0.0.1:7000/admin
2. Введите логин и пароль суперпользователя.
3. В открывшейся административной панели мы можем просматривать, создавать и редактировать записи из всех таблиц базы данных.
Предполагается, что из административной панели будет вводиться информация о специальностях и врачах.
4. Для корректной работы приложения со стороны пользователя советуем ввести информацию о нескольких специальностях и врачах, которые будут отображаться пользователю.

------------------
ПОЛЬЗОВАТЕЛЬ
------------------

1. Зайдите по адресу  http://127.0.0.1:7000/
2. Зарегистрируйтесь в приложении (форма регистрации появится после нажатия на кнопку "Registration").
3. Войдите в приложение (на форму авторизации Вы будете автоматически перенаправлены после регистрации).
Если пользователь уже был зарегистрирован, он может воспользовать кнопкой "Login" на стартовой странице.
4. После авторизации будут доступны опции просмотра каталога и заказа книг, а также входа в личный кабинет.
5. Для заказа книги:
5.1. Нажмите кнопку "Catalog", псоле чего Вы будете переадресованы на страницу, где можно выбрать просмотреть каталог доступных книг и выбрать книги, которые Вы хотите заказать.
5.2. Выберите книги для заказа и подтвердите выбор, после этого Вы будете перенаправлены на страницу оформления заказа.
5.3. Нажмите кнопку "Подтвердить заказ", после чего Вы будете перенаправлены в личный кабинет.
6. Для входа в личный кабинет нажмите кнопку "Cabinet", после чего Вы будете перенаправлены на страницу личного кабинета, на которой можно просмотреть историю заказов книг. 
7. С целью выхода из приложения нужно нажать кнопку "Logout" на главной странице или в личном кабинете.
